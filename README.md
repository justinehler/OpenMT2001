# OpenMT2001

OpenMT2001 is an open-source project for documenting, studying, and eventually publishing clean-room tooling around *Monopoly Tycoon* (2001) file formats and preservation workflows.

This repository currently contains only public project metadata. Source code and technical artifacts will be published after legal/IP review confirms that no proprietary game assets, extracted files, or redistributable third-party content are included.

## Goals

- Document the original game's archive, script, texture, model, and configuration formats.
- Build tooling that helps inspect and validate legally owned local installations.
- Keep proprietary assets and extracted game data out of the public repository.
- Publish implementation work under the MIT License once it is ready for public release.

## Status
